Para correr a geração das caracteristicas S, S0 e I_inv 
fazer python nome_file.py "diretoria_input" "diretoria_output" 1 
 - A opção (1) indica que é para gerar as caracteristicas
 - diretoria_input deve ter dentro dela as pastas test e training
   As pastas test e training deve obeceder Às sub pastas e nomes dwe ficheiros da base de dados DRIVE
 - A diretoria_output pode ser uma qualquer. Dentro dela serão criadas as pastas Treino e Teste.
   
exemplo: 
    python .\main.py "./DRIVE/" "./Caracteristicas/" 1


Para realizar segmentação baseada nas imagens S: 
fazer python nome_file.py "diretoria_input" "diretoria_output" 2 
 - A opção (2) indica que é para segmentar por S
 - diretoria_input deve ter dentro dela a pasta com as imagens S, usando os nomes das imagens
 geradas no passo anterior
 - A diretoria_output indica a pasta onde serão guardadas as imagens segmentadas 
exemplo:
    python .\main.py "./Caracteristicas/Treino/S_img/" "./Segmentacao_S/" 2
